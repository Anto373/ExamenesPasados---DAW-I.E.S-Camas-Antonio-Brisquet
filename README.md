# Repositorio de Exámenes de Desarrollo de Aplicaciones Web (DAW)

Este repositorio está dedicado a recopilar los exámenes que se han realizado en el curso de **Desarrollo de Aplicaciones Web (DAW)**. Los exámenes han sido proporcionados por un alumno anónimo. Las carpetas pueden contener nombres de docentes, pero sin los apellidos.

## Asignaturas incluidas

Los exámenes se dividen por asignaturas, que son las siguientes:

- **Bases de datos**
- **Entornos de desarrollo**
- **Formación y orientación**
- **Lenguaje de marcas**
- **Programación**
- **Sistemas informáticos**

## Diferencias entre los exámenes

Los exámenes aquí subidos pueden variar en contenido y dificultad. Algunos no se han podido guardar debido a la complejidad para su almacenamiento. Por ello, puede resultar difícil a veces desenvolverse en el repositorio y buscar el contenido. Sin embargo, he creado este repositorio con la intención de facilitar la experiencia de estudio y preparación para los exámenes de DAW.
